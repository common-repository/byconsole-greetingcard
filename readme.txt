== Description ==



Let you customers choose a complimentary greetings card on checkout page.



= Upload up to four greetings cards as pdf or image in back-end  =

Upload up to four greetings cards in back-end settings page and let your customers to choose one from your uploaded greetings cards



= Preview the chosen greetings card on checkout page =

Your customers can choose what greetings card they want to send to their loved one along with the gift(ie; birthday flower boutique order on your store)



= See the chosen greetings card on admin side order details page =

When you process the order you can see what greeting cards was chosen by customer.



= Show the chosen greetings card on customer side order details page =

Selected greetings card will be shown on customers order details page in front-end.



= Show the selected greeting card on new order mail =

Admin can see the selected greetings card on new order notification mail.



= Show the selected greeting card on customer's order confirmation mail =

Customers can see the selected greetings card on order confirmation mail.

== Installation ==



1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly by uploading zip through 



"Upload Plugin" button in "Plugins" -> "Add New" screen of wp-admin area.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Use the Wp admin menu -> Order Delivery Time management screen to configure the plugin

4. Use the Wp admin menu -> Appearance -> Widgets -> Order delivery time widget and place it on your desired sidebar



OR



Search for "Complimentary greetings card for WooCommerce" in "Plugins" -> "Add New" screen within your WordPress wp-admin area and click install button





== Frequently Asked Questions ==



= I have uploaded 4 greetings cards in back-end plugin settings page, but not showing all four greetings card on checkout page =



Please make sure all the uploaded greetings cards have different file name than each other, and you have checked the checkbox "Enabled" for cards you wish to 



show on checkout page.



= Radio button have no label =



Make sure you have put label text for each of enabled greetings card in plugin settings page.



= Greetings card preview not appearing =



Make sure you have placed only full URL of the pdf or image file, if your greetings card link was placed using "Add Media" button then review if it is wrapped 



with <a href="xxx"....>...</a>, if so remove the <a></a> and put only full path of the file.
